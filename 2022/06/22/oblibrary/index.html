obsidian创建个人图书管理库

Author: LaiJiahao Date: June 22, 2022  14:28:55

在学习obsidian的过程中，想要把一直没有实现的图书管理功能实现，于是这样的一个教程

最终效果：

本文相关文件：https://mrhuanhao.lanzoum.com/iFZm006ssl0j

需要的插件

  1. dataview
  2. quickadd
  3. templater
  4. buttons

在正式开始之前，确保你设置好了模板插件、templater插件并且创建一个文件夹存放数据

- 读书笔记
    - 我的书架

读书笔记模板

书名：<% tp.file.title %>

状态：<% tp.system.suggester(["在读","已读","想读"],["📖","☑️","📕"]) %>

作者：{{VALUE:author}}

出版：{{VALUE:publish}}

出版时间：{{VALUE:publishyear}}

开始时间：<% tp.date.now() %>

读完时间：

总页数：{{VALUE:pagecount}}

已读页数：[0]

我的评级：⭐⭐⭐⭐⭐

重读次数：0

介质：<% tp.system.suggester(["纸质","电子书"],["纸质","电子书"]) %>

封面：{{VALUE:douban_url}}

豆瓣评分：{{VALUE:rating}}

ISBN：{{VALUE:isbn}}

tag：

其他内容自行设置

配合quickadd

  1. 添加新的quickadd，名字为读书笔记，类型为template
  2. 选中刚刚的读书笔记模板
  3. create in floder 设置读书路径
  4. open 且 focus
  5. 打开小闪电

设置quickadd

  1. 进入管理宏界面

  2. 创建一个新的宏为：添加读书笔记

  3. 进入该宏的配置界面
    先添加一个js脚本，这个脚本位置随意，在obsidian里面就可以了，可以放在我的读书笔记文件夹下面

use scripts 会自动识别，然后点击add即可

脚本下载地址：https://mrhuanhao.lanzoum.com/iFZm006ssl0j

  1. 添加template，流程和刚刚一个但是需要设置file name
    数据为{{VALUE:bookname}}

减低template按钮，然后进行设置

  1. 删除刚刚的quickadd，新建新的quickadd，类型为macro
    然后设置

  2. 为新的quickadd设置一个快捷键

  3. 接下来重新打开obsidian

这时候按下快捷键就可以生效了

如何隐藏标签

把下面这个css代码放进snippets随便取个名词

.frontmatter-container {
    display: none;
}
