dataview的语法

list 从库中按照一定的条件，以列表的形式展示 / Table 以表格形式展示
from "路径获取" / #获取某个标签
where 进一步过滤
sort 按照某种方式排序

例子：

list "文件的修改日期"+file.mtime / Table without id file.link as 书名 file.ctime as 创建时间,file.mtime as 修改时间 #as代表修改列名
from
where file.name !="读书笔记模板"  #意思是不包含这个文件名
sort file.cady,file.name desc/asc #以创建日期排序 日期相同则按照文件名顺序 降序/升序
